# Regression Models

🚀 This repository is a collection of regression models. 🚀

## 🧑‍🎓 1. Placement Prediction

To predict the package (in LPA) based on the `CGPA` of the student, under the assumption that every student gets placed.

Dataset used:
- [placement_data.csv](data/placement_data.csv)

Models used :
- Linear Regression Model

## 🏠 2. House Rate Prediction

To predict the rate at which a house will be sold based on various factors regarding the house.

Dataset used :
- [real_estate.csv](data/real_estate.csv)

Models trained : 
- Linear Regression Model
- Linear Regression Model with Lasso Regularization
- Linear Regression Model with Ridge Regularization

## 🚗 3. Car Purchase

To predict the price range in which a person buys cars based on his financial information.

Dataset used:
- [car_purchasing.csv](data/car_purchasing.csv)

Model trained :
- Linear Regression Model

## 🍷 4. Wine Quality Analysis

To predict the qualify of wine on a scale of 10, based on the quantitative and qualitative analysis data on the wine.

Dataset used:
- [wine_quality_data.csv](data/wine_quality_data.csv)

Model trained :
- Linear Regression Model

## 📈 5. Polynomial Dataset

To understand the limitations of linear regression model on a dataset where output label is poylnomially dependent upon the input feature, and understand how artificial neural network models perform on the dataset. 

Dataset used:
- [poly.csv](data/poly.csv)

Models trained:
- Linear Regression Model
- A two layer neural network
